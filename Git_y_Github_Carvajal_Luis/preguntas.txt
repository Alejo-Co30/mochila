¿Quien inventó el sistema de control de versión Git y por qué?
Fue inventado por Linus Torvalds en el 2005, lo motivo la necesidad de un sistema que permitiera compartir archivos y documentos entre desarolladores, sin la necesidad de alterar los codigos centrales y poder ofrecer versiones del mismo.

¿A quién pertenece actualmente Github y por qué?
Actualmente el sistema de versiones le pertenece a Microsoft. Reconocieron en este programa una gran capacidad para el almacenamiento de proyectos.

¿Hay otra forma que no sea la terminal para trabajar con Github?
Se puede usar desde su página principal, pero sus funciones a grandes rasgos, solo puede ser usada en la terminal.
